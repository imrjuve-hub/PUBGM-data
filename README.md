# PUBG Match Microsite

Static single-file site for Vercel.

## Deploy on Vercel
1. Create a new GitHub repo and add `index.html` at the root (this file).
2. In Vercel, click **New Project** â†’ **Import** your repo.
3. Framework preset: **Other**. Build command: leave empty. Output directory: `/`.
4. Deploy. Open the URL and upload a match JSON to view leaderboards, MVP, and awards.

No server needed. Everything runs in the browser.
