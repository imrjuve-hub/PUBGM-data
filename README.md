# PUBG Match Microsite - Multi Match

Static single-file site for Vercel. Supports uploading multiple match JSON files, aggregates an overall team leaderboard, and shows player stats per team.

## Deploy on Vercel
1. Create a new GitHub repo and add `index.html` at the root.
2. In Vercel, click New Project -> Import your repo.
3. Framework preset: Other. Build command: leave empty. Output directory: `/`.
4. Deploy. Open the URL and upload one or more match JSON files.

Everything runs in the browser.
